Proyecto: App de pedidos
Tecnologías: Dart, Flutter

La aplicación de Pedidos es una herramienta diseñada para simplificar y agilizar el proceso de gestión de pedidos en un entorno comercial. Con su capacidad para mantener una base de datos detallada de productos disponibles, así como para registrar pedidos individuales con información sobre los productos solicitados y las cantidades requeridas, la aplicación ofrece una solución al negocio que maneja ventas y distribución de productos. Además, su funcionalidad para generar automáticamente pedidos de reposición al proveedor, junto con la opción de compartir estos pedidos en formato PDF a través de WhatsApp, añade un nivel adicional de conveniencia y eficiencia a la operación, Este proyecto se llevó a cabo en colaboración con Oscar Ortiz.

Proyecto: Facturacion e Inventario
Tecnologías: java, java swing, MySQL

El programa de facturación e inventario tiene como objetivo principal demostrar las habilidades y conocimientos técnicos del desarrollador en el desarrollo de aplicaciones Java para la gestión de negocios. Se establecen claros objetivos de alcance que incluyen la generación de facturas y comprobantes de crédito fiscal, la actualización automática del inventario al facturar, la consulta de inventario en tiempo real, acceso seguro mediante autenticación de usuarios, un diseño intuitivo, y la utilización de una base de datos relacional (MYSQL) con diseños de tablas que incluyen store procedures, triggers y views. Se prioriza el rendimiento del sistema, su escalabilidad y disponibilidad, junto con la capacidad de generar reportes tanto en pantalla como en Excel, y la posibilidad de editar, actualizar, anular y eliminar facturas, manteniendo la coherencia con el stock de productos en la base de datos.

Proyecto: Calculadora de costos de importacion
Tecnologías: flutter / Dart

Desarrollamos una aplicación sencilla que ayuda a los usuarios a estimar costos de importación utilizando fórmulas matemáticas predefinidas en la programación. Este proyecto se llevó a cabo en colaboración con Oscar Ortiz. La aplicación utiliza parámetros como cajas por contenedor, unidades por caja, precio por caja, costo del flete y otros gastos relacionados. Además, la aplicación también tiene en cuenta los impuestos de importación, como el DAI y el IVA, para proporcionar una estimación precisa de los costos totales de importación.

Proyecto: App Full Stack
Tecnologías: Spring Boot, React y MySQL

Esta aplicación web es una solución full stack que incluye un sistema de gestión de datos (CRUD) completo. Está dividida en dos partes: un frontend desarrollado con React y un backend desarrollado con Spring Boot. El frontend ofrece una interfaz intuitiva para el usuario, mientras que el backend proporciona la lógica de negocio y el acceso a los datos. Los datos se almacenan y gestionan mediante una base de datos MySQL. Esta aplicación permite realizar operaciones de creación, lectura, actualización y eliminación de datos de manera eficiente y segura.
